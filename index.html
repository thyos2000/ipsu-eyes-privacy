DevLoc is a secure internal monitoring application intended exclusively for authorized operational personnel.

Information We Process

The application processes the following operational data:

  • Work email address (for authentication)
  • Authentication verification codes (temporary)
  • Device identifiers for secure session management
  • Push notification token (to deliver alerts)
  • Infrastructure monitoring data (device name, IP address, status)

No personal profile data, contacts, photos, or personal files are accessed or collected.

How Information Is Used

  • Authenticate authorized users
  • Deliver monitoring alerts
  • Maintain secure sessions
  • Provide operational situational awareness

Data Storage

Authentication tokens are securely stored on the device using the iOS secure keychain. Verification codes are temporary and automatically expire. No user data is sold, shared, or used for advertising.

Data Sharing

Data is never shared with third parties. All communication occurs only between the application and the official monitoring server.

Access Restriction

The application is restricted to pre-approved organizational accounts. Unauthorized users cannot register or access system data.

Security

All communications are encrypted using HTTPS. Push notifications contain operational alerts only and no personal information.

Contact

For operational or privacy questions contact the system administrator.
